  1. Use the mater method to give tight asymptotic bounds for the following recurrences:
    1. $T(n) = 2T(n/4) + 1$

      We have that $a = 2$, $b = 4$, $f(n) = 1$, $n^{\text{log}_ba} = n^{\text{log}_42} = n^{0.5} = \Theta(\sqrt{n})$.
      $f(n) = O(1) = O(n^{0.5 - \epsilon}) = O(n^{0.5 - 0.5}) = O(n^0) = O(1)$ where $\epsilon = 0.5$, then we use case 1. \begin{align} T(n) &= \Theta(n^{\text{log}_ba}) \\ T(n) &= \Theta(n^{\text{log}_42}) \\ T(n) &= \Theta(n^0.5) \\ T(n) &= \Theta(\sqrt{n}) \end{align}

    2. $T(n) = 2T(n/4) + \sqrt{n}$

      We have that $a = 2$, $b = 4$, $f(n) = \sqrt{n}$, $n^{\text{log}_ba} = n^{\text{log}_42} = n^{0.5} = \Theta(\sqrt{n})$.
      $f(n) = \Theta(\sqrt{n})$, then we use case 2. \begin{align} T(n) &= \Theta(n^{\text{log}_ba}\text{lg }n) \\ T(n) &= \Theta(n^{\text{log}_42}\text{lg }n) \\ T(n) &= \Theta(n^0.5\text{lg }n) \\ T(n) &= \Theta(\sqrt{n}\text{lg }n) \end{align}

    3. $T(n) = 2T(n/4) + n$

      We have that $a = 2$, $b = 4$, $f(n) = n$, $n^{\text{log}_ba} = n^{\text{log}_42} = n^{0.5} = \Theta(\sqrt{n})$.
      $f(n) = \Omega(n) = \Omega(n^{0.5 + \epsilon}) = \Omega(n^{0.5 + 0.5}) = \Omega(n^1) = \Omega(n)$ where $\epsilon = 0.5$, then we use case 3. \begin{align} T(n) &= \Theta(n) \end{align}

    4. $T(n) = 2T(n/4) + n^2$

      We have that $a = 2$, $b = 4$, $f(n) = n^2$, $n^{\text{log}_ba} = n^{\text{log}_42} = n^{0.5} = \Theta(\sqrt{n})$.
      $f(n) = \Theta(n^2) = \Omega(n^{0.5 + \epsilon}) = \Omega(n^{0.5 + 1.5}) = \Omega(n^2)$ where $\epsilon = 1.5$, then we use case 3. \begin{align} T(n) &= \Theta(n^2) \end{align}

  2. Professor Caesar wishes to develop a matrix-multiplication algorithm that is asymptotically faster than Strassen's algorithm. His algorithm will use the divide-and-conquer method, dividing each matrix into pieces of size $n/4 \times n/4$, and the divide and combine steps together will take $\Theta(n^2)$ time. He needs to determine how many subproblems his algorithm has to create in order to beat Strassen's algorithm. If his algorithm creates $a$ subproblems, then the recurrence for the running time $T(n)$ becomes $T(n) = aT(n/4) + \Theta(n^2)$. What is the largest integer value of $a$ for which Professor Caesar's algorithm would be asymptotically faster than Strassen's algorithm?

    Strassen's algorithm runs in $T(n) = \Theta(n^\text{lg }7)$ we need to find an $a$ such that $n^{\text{log}_4a} \lt n^{\text{lg }7}$ \begin{align} n^{\text{log}_4a} = n^{\text{lg }7} \\ n^{\text{lg }\sqrt{a}} = n^{\text{lg }7} \\ \sqrt{a} = 7 \\ a = 7^2 \\ a = 49 \end{align} The largest number of subproblems must is 48 before Strassen's algorithm beats Professor Caesar's algorithm.
